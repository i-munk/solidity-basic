# Solidity Basic Study 📚

Solidity 스마트 컨트랙트 기본 문법을 학습하며, 직접 작성한 예제들을 Hardhat + Ignition을 활용해 배포했습니다.

---

## 🔧 기술 스택

- Solidity ^0.8.x
- Hardhat
- Hardhat Ignition (배포 자동화)
- TypeScript
- VSCode

---

## 📂 디렉토리 구조

```bash
/contracts         # 스마트 컨트랙트 코드 (.sol)
/ignition/modules  # Ignition 배포 모듈 (.ts)
/test              # (선택) 테스트 코드
```
